# :iphone: SwAg

Apple Developer Academy @ POSTECH, NC1 Project

![iOS](https://img.shields.io/badge/Swift-iOS-51a9e8?logo=Swift)

스위프트 코드와 함께 자료구조와 알고리즘을 배워보아요! <br>
자료구조와 알고리즘의 애니메이션을 스텝 바이 스텝으로 확인하고 스위프트 코드로 확인할 수 있습니다. ~~코드 복사는 덤~~

## :pushpin: Features

- 직관적인 Visualization
- 라이트/다크 모드 따라 색상 변환
- Swift 소스코드 보기/복사 기능
- 다국어(영어/한국어) 지원
